# VK Stalker Bot

Telegram бот для мониторинга онлайн статуса пользователей VK.

## Возможности

- Отслеживание онлайн/офлайн статуса пользователей VK
- Подсчет общего времени онлайна
- Отдельные базы данных для каждого Telegram пользователя
- Уведомления о появлении/выходе из сети
- Тихий режим (без уведомлений)

## Команды

- `/start` - Начать работу и добавить пользователя
- `/add` - Добавить еще одного пользователя
- `/remove` - Удалить пользователя из отслеживания
- `/status_main` - Показать статус основного пользователя
- `/status_all` - Показать статус всех пользователей
- `/on` - Включить уведомления
- `/off` - Выключить уведомления (тихий режим)
- `/help` - Помощь

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте токены в файле `bot_working_fixed.py`
4. Запустите: `python bot_working_fixed.py`

## Переменные окружения

Для работы на хостинге Amvera:

- `VK_TOKEN` - Токен VK API
- `TG_TOKEN` - Токен Telegram Bot
- `PYTHONUNBUFFERED` - Установите в `1` для корректного вывода логов

## Структура баз данных

Каждый Telegram пользователь имеет отдельную базу данных:
- `database_{user_id}.json` - Персональная база данных

## Особенности

- Полная изоляция данных между пользователями
- Автоматическое создание баз данных
- Резервное копирование и восстановление
- Обработка ошибок VK API
- Асинхронная работа с Telegram API
